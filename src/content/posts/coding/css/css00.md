# 2. `ease`
| 速度曲線       | 效果描述                 |
|----------------|--------------------------|
| **linear**     | 均速，無變化             |
| **ease**       | 開始慢 → 加速 → 結束慢  |
| **ease-in**    | 開始慢 → 加速           |
| **ease-out**   | 開始快 → 減速結束       |
| **ease-in-out**| 開始慢 → 加速 → 結束慢  |
